# Life Path Number 生命灵数小测试

#### https://tomoko-tiba.github.io/an-demo/

### 使用指南

1. 输入8位数生日，如19990120
2. 查看你的灵数解释，左滑右滑切换你的关键词，点击底部button（⬇️）查看详细说明。
